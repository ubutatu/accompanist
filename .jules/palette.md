# Palette's Journal - Critical UX/Accessibility Learnings

## 2025-05-15 - [Visual Feedback for Permissions]
**Learning:** Permission request UIs often lack clear visual feedback for both the requirement and the success states. Using icons like `CameraAlt` for the request and `Done` for the granted state provides immediate visual context that transcends language barriers.
**Action:** Always include visual indicators (icons/illustrations) and descriptive button labels (e.g., "Request camera permission" vs "Request permission") in permission-related flows to improve both UX and accessibility.
